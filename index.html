¿Quienes somos y que ofrecemos?

Somos un grupo de tres personas que estamos estudiando el grado superior de administracion de sistemas informaticos en red. Para este trabajo los compañeros Carlos Mateo, Ignacio Delosriscos y Samuel Cabrera hemos creado un script, y esta pagina web donde explicamos en que consiste nuestro trabajo.

Script de powershell


Este script que hemos creado sirve para visualizar todos los procesos del ordenador y poder pararlos o iniciarlos indicando exactamente cual.



Estos son principalmemte los comandos que hemos usado


la funcion verproceso se compone de una linea que nos devuelve informacion del proceso que pidas prorrateado.


la funcion pararproceso la primera linea pregunta el nombre del proceso y lo mete dentro de una variable. en la segunda linea se ejecuta la funcion para que nos de el proceso que queremos con todas sus caracteristicas. en la tercera linea pedimos la id de dicho proceso y lo metemos n una variable. en la cuarta linea le decimos al proceso que se pare con la id antes pedida.


la funcion iniciarproceso En la primera linea pedimos el nombre y lo pasamos a una variable. en la segunda linea iniciamos el proceso antes pedido.


Realizamos un do/while para hacer una estructura repetitiva y que el menu realize las funciones en el orden indicado. despues de elegir una de las opciones y realizarla nos volvera a saltar el menu para preguntarnos si queremos ejecutar el programa de nuevo.


Dentro del do introducimos un switch el cual es el encardado de hacer funcionar el menu. Realiza primero una pregunta para que eligas una respuesta y posteriormente realice la funcion del numero asiganado.
