Script de powershell


Este script que hemos creado sirve para poder indentificar y clasificar un archivo o programa que tengamos en nuestro equipo de manera que, sin necesidad de ejecutarlo y poder contaminar el equipo, nos dsa informacion sobre si ese archivo es un virus o perjudicial para el ordenador o si en caso contrario no hay peligro de mantenerlo en el equipo



Estos son principalmemte los comandos que hemos usado


El cmdlet Get-FileHash calcula el valor de hash para un archivo utilizando un algoritmo de hash especificado.


El Hash es un valor único que corresponde al contenido del archivo. En lugar de identificar el contenido de un archivo por su nombre de archivo, extensión u otra designación, un hash asigna un valor único al contenido de un archivo.


El SHA1 se trata de un algoritmo de encriptacion que separa en bloques de 512 bits y además para cada bloque añade 80 vueltas con unos vectores llamados ABCDE y se mezclan con el siguiente y así sucesivamente hasta hacer un resumen de 160 bits de todo el documento.


El MD5 es un algoritmo de reducción criptográfico de 128 bits ampliamente usado. Uno de sus usos es el de comprobar que algún archivo no haya sido modificado. a codificación del MD5 de 128 bits es representada típicamente como un número de 32 símbolos hexadecimales.
